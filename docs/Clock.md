# Clock

时钟，用于执行指定周期的任务或倒计时任务

拥有以下几个特点：

1. 需要指定执行周期；
2. 可以不间断的执行同一个任务
3. 任务逻辑相对独立，也可能是多种逻辑的混合物，难以归类
4. 可能是临时性的
5. 可以被暂停或者继续执行任务

> ### 约定 - 时钟的定义
>
> `name`			时钟的名字
>
> `type`			时钟的类型，封装了指定逻辑代码
>
> `initParams`		初始参数表，用于逻辑计算，**必须为字面量属性对象，用于存储在Memory中**
>
> `func`			时钟执行时的具体逻辑，因为当runtime中的时钟对象被清理后，需要从Memory中恢复			
>
> ​				数据，所以需要硬编码func逻辑作为不同type的时钟
>
> `tick`			周期，每次执行任务的时间间隔
>
> `active`			状态，表示是否激活/启动
